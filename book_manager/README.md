# Book Manager - PyQt5 Application
Простое приложение для отображения каталога книг из файла данных.
## Особенности
- Загрузка данных из текстового файла
- Отображение в табличном виде
- ГУИ на PyQt5
## Требования
- Python 3.7+
- PyQt5
